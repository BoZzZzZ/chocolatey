There are many ways to purchase credits on Habbo!

Buy a code with Cherry Credits

Cherry Credits is a virtual currency that you can use to buy Habbo Credits.

  1. Log into Cherry Credits. (If you don't already have an account, create one first. You can then top up that account in stores, at kiosks or even online - check the topping-up instructions for Cherry Credits.)

  2. Buy Habbo Credits on the Cherry Credits website.

  3. Once you've bought the Habbo credits, you'll get a code. Come back to this page and enter the code in the voucher code field. (Remember to log in first.)

Buy a code with credit card or PayPal

Habbo codes bought on GameTimeZone and PCGameSupply are valid in all Habbo hotels! You can buy one with credit cards or PayPal. Once you have the code, come back to this page and enter the code in the voucher code field! (Remember to log in first.)
